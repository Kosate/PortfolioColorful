Chula Pop Bus 3.0
Android Developer (Main) & Designer (Main)

แอพพลิเคชั่น Chula Pop Bus เป็นแอพพลิเคชั่นเพื่ออำนวยสะดวกในการใช้รถปรับอากาศพิเศษ (ปอพ. หรือเรียกว่าป็อบ) ของจุฬาลงกรณ์มหาวิทยาลัย

ตัว App จะ Tracking ข้อมูล GPS ที่ติดบนรถป็อบแล้วแสดงผลลงบน Application อีกทั้งยังมีระบบค้นหาอาหารภายในจุฬาลงกรณ์มหาวิทยาลัย รวมไปถึงแสดงเส้นทางโดยสารของรถป็อบ ซึ่งข้อมูลทุกอย่างนั้นเป็น Dynamic หมด ข้อมูลทุกอย่างดึงมาจาก Database และยังมีการ caching ไว้ในเครื่องด้วย กรณีที่เครื่อง offline ก็สามารถใช้งานได้โดยไม่ต้องต่อ Internet

ผมได้รับผิดชอบพัฒนาตัวแอพพลิเคชั่นเวอร์ชั่น 3.0 ซึ่งเป็นทั้งคนพัฒนาและดีไซน์ด้วยสำหรับงานนี้ สำหรับ Front-end ผมใช้ Java พัฒนา Android Studio ตามปกติ และเชื่อมต่อกับ Database โดยใช้ Parse API (Facebook เป็นเจ้าของ) ซึ่งขณะนี้มีแผนในอนาคตว่าจะย้ายไปใช้ Firebase ไม่ก็ host database เองในเร็วๆนี้ครับ
