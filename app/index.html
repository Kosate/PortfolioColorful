Chula Pop Bus 3.0
Android Developer (Main) & Designer (Main)

แอพพลิเคชั่น Chula Pop Bus เป็นแอพพลิเคชั่นเพื่ออำนวยสะดวกในการใช้รถปรับอากาศพิเศษ (ปอพ. หรือเรียกว่าป็อบ) ของจุฬาลงกรณ์มหาวิทยาลัย

ตัว App จะ Tracking ข้อมูล GPS ที่ติดบนรถป็อบแล้วแสดงผลลงบน Application อีกทั้งยังมีระบบค้นหาอาหารภายในจุฬาลงกรณ์มหาวิทยาลัย รวมไปถึงแสดงเส้นทางโดยสารของรถป็อบ ซึ่งข้อมูลทุกอย่างนั้นเป็น Dynamic หมด ข้อมูลทุกอย่างดึงมาจาก Database และยังมีการ caching ไว้ในเครื่องด้วย กรณีที่เครื่อง offline ก็สามารถใช้งานได้โดยไม่ต้องต่อ Internet

ผมได้รับผิดชอบพัฒนาตัวแอพพลิเคชั่นเวอร์ชั่น 3.0 ซึ่งเป็นทั้งคนพัฒนาและดีไซน์ด้วยสำหรับงานนี้ สำหรับ Front-end ผมใช้ Java พัฒนา Android Studio ตามปกติ และเชื่อมต่อกับ Database โดยใช้ Parse API (Facebook เป็นเจ้าของ) ซึ่งขณะนี้มีแผนในอนาคตว่าจะย้ายไปใช้ Firebase ไม่ก็ host database เองในเร็วๆนี้ครับ

Archievement
Thailand Olympiad in International (TOI)
คอมพิวเตอร์โอลิมปิกระดับชาติ

คอมพิวเตอร์โอลิมปิกระดับชาติเป็นการแข่งขันของนักเรียนระดับชั้นมัธยมผู้ที่สนใจด้านการเขียนโปรแกรม

ลักษณะของการแข่งขันจะเป็นลักษณะโจทย์ทั้งหมด 6 ข้อ แบ่งเป็น 2 วัน วันละ 3 ชั่วโมงและ 3 ข้อ รูปแบบของโจทย์นั้นจะต้องหยิบเอา Data Structure, Algorithm ที่ดีมาใช้

ซึ่งเนื้อหาครอบคลุมตั้งแต่ Sorting, Searching, Recursive, Greedy, Dynamic Programming, Hashing, Linked List, Tree, Heap, Graph และอื่นๆอีกมากมาย โดยจะต้องเขียนภาษา C, C++ เพื่อรับ Input นำไปประมวลผลและส่ง Output ออกมา ซึ่งจะต้องเขียนให้ Optimize ที่สุด (เพราะโจทย์กำหนดเวลาและ Memory ของโปรแกรม) เพื่อแก้ปัญหาในแต่ละข้อ

ซึ่งผมได้ร่วมเข้าแข่งขันทั้งหมด 3 ครั้ง ครั้งแรกตั้งแต่ ม.3 โดยได้เหรียญทองแดง (TOI ครั้งที่ 7) เหรียญทอง (TOI ครั้งที่ 8) และเหรียญเงินตาม (TOI ครั้งที่ 9) ลำดับ

ACM-ICPC
International Collegiate Programming Contest

ACM-ICPC เป็นรายการเขียนโปรแกรมระดับชั้นมหาวิทยาลัย ซึ่งเป็นการแข่งขันที่ต้องใช้ Data Structure และ Algorithm ที่ดีในการแก้ปัญหา รูปแบบการแข่งจะลักษณะคล้ายคลึงกับ TOI แต่มีความยากมากกว่าหลายเท่า

ผมได้เข้าร่วมการแข่งขันในปี 2014 และ ปี 2015 ซึ่งได้เข้าลึกไปจนถึงรอบระดับเอเชีย

ในปี 2014 ทีมของผม CPCUBUG ได้ลำดับที่ 46 และปี 2015 ได้ลำดับที่ 21 ในชื่อทีม CUCP NoScore จากผู้เข้าแข่งขันร่วมทั้งหมดเกือบ 90 ทีมที่ผ่านการคัดเลือกทั่วเอเชีย

Codehew Hackathon

รายการ Codehew Hackathon เป็นการแข่งขันเขียนโปรแกรมโจทย์เพื่อแก้ปัญหา สนับสนุนโดย Wongnai

ลักษณะการแข่งขันจะเป็นเชิง Practical มากกว่า Competitive ไม่เหมือน 2 รายการที่ผ่านข้างต้น โจทย์จะเป็นลักษณะที่เล่นเกี่ยวกับ Data เช่น ค้นหาคำ, หาข้อมูลที่ดีที่สุดจากที่มีให้ อาจจะมาจากการอ่าน/เขียนไฟล์ อีกทั้งยังครอบคลุมถึง Image Processing, AI เกม XO และยังมีโจทย์เชิง Algorithm อีกด้วย

การแข่งขันใช้เวลาประมาณครึ่งวัน ซึ่งทีมของผม (2 คน) ได้ลำดับที่ 3 จากการแข่งขันในทีมชื่อว่า NULL สำหรับตัวโค้ดที่ใช้แก้ปัญหาอยู่ที่ Github ด้านล่างครับ
