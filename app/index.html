Archievement
Thailand Olympiad in International (TOI)
คอมพิวเตอร์โอลิมปิกระดับชาติ

คอมพิวเตอร์โอลิมปิกระดับชาติเป็นการแข่งขันของนักเรียนระดับชั้นมัธยมผู้ที่สนใจด้านการเขียนโปรแกรม

ลักษณะของการแข่งขันจะเป็นลักษณะโจทย์ทั้งหมด 6 ข้อ แบ่งเป็น 2 วัน วันละ 3 ชั่วโมงและ 3 ข้อ รูปแบบของโจทย์นั้นจะต้องหยิบเอา Data Structure, Algorithm ที่ดีมาใช้

ซึ่งเนื้อหาครอบคลุมตั้งแต่ Sorting, Searching, Recursive, Greedy, Dynamic Programming, Hashing, Linked List, Tree, Heap, Graph และอื่นๆอีกมากมาย โดยจะต้องเขียนภาษา C, C++ เพื่อรับ Input นำไปประมวลผลและส่ง Output ออกมา ซึ่งจะต้องเขียนให้ Optimize ที่สุด (เพราะโจทย์กำหนดเวลาและ Memory ของโปรแกรม) เพื่อแก้ปัญหาในแต่ละข้อ

ซึ่งผมได้ร่วมเข้าแข่งขันทั้งหมด 3 ครั้ง ครั้งแรกตั้งแต่ ม.3 โดยได้เหรียญทองแดง (TOI ครั้งที่ 7) เหรียญทอง (TOI ครั้งที่ 8) และเหรียญเงินตาม (TOI ครั้งที่ 9) ลำดับ
